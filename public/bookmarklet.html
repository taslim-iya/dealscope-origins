How to use

1Drag the purple button above to your bookmarks bar

If your bookmarks bar is hidden, press Ctrl+Shift+B (Windows) or Cmd+Shift+B (Mac)

2Go to any business listing site and search

Visit any of the sites below and browse the listings page

3Click the bookmarklet

It extracts all visible listings on the page and downloads a JSON file

4Repeat for each page

Navigate to page 2, 3, etc. and click the bookmarklet again on each page. Send me all the JSON files.
